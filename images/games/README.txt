GAME ICONS FOLDER
=================

Save your game icon images in this folder. The Games window shows
one icon per game and looks for these files:

  doom.png
  snake.png
  tictactoe.png
  textadventure.png

Square PNG images (for example 128 x 128) work best.

If an icon image is missing, the Games window falls back to a simple
emoji icon for that game, so the page still works fine without them.

To rename, add, or remove games, edit the GAMES list near the top of
script.js.
