IMAGES FOLDER
=============

DESKTOP WALLPAPER
-----------------
Save your desktop background photo in this folder as:

  wallpaper.jpg

That is the file the website looks for (see the #desktop rule in
style.css). To change the wallpaper later, just replace this file
with another image of the same name.

For the mountain/lake photo you picked: save it here as
wallpaper.jpg, then hard-refresh the page.

TIPS
----
- A wide, landscape-shaped image works best (the photo you chose,
  about 2600 x 1462, is ideal).
- .jpg keeps the file size small; .png also works if you rename the
  reference in style.css.
- If the file is missing, the desktop falls back to a blue-to-green
  gradient, so the page still works.
